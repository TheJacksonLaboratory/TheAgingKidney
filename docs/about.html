About

The work contained here has been collaboratively conducted by the Korstanje Lab (The Jackson Laboratory), Churchill Lab (The Jackson Laboratory), and the Gygi Lab (Harvard Medical School).

We used the Diversity Outbred mice, a powerful genetic resource for high resolution mapping, to model both transcriptomic and proteomic changes that occur during renal aging in a genetically diverse population.

This online resource provides source code used to conduct analyses and produce figures and tables that appear in our publication.
